# MIA NLP Challenge
## Compound Nouns
### Installation
`pip install -r requirements.txt`
### Usage
To run the application, run:
`uvicorn main:app`
### Tests
All tests are under the folder **/tests**. In order to run the tests:
* `cd /tests`
* `pytest` (_to run all the tests_) or
* `pytest <FILE_NAME>` (_to run each test separately_)
